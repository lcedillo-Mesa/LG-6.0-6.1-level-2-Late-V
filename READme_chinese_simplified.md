# 评估 2 — CSS 选择器与盒模型
# Assessment 2 — CSS Selectors & Box Model

---

## 你的任务 / Your Task

你将看到一个作品集页面的渲染效果图。

You will be given a rendered mockup of a portfolio page.

你的任务是编写 CSS 规则，使你的页面尽可能接近效果图。

Your job is to write the CSS rules that make your page look as close to the mockup as possible.

打开 `styles.css`，在 `/* Write your code below this line */` 这一行下面写你的规则。

Open `styles.css` and write your rules below the line that says `/* Write your code below this line */`.

---

## 你需要展示的内容 / What you need to demonstrate

**选择器 / Selectors (LG 6.0)**
- 使用类选择器来定位单个元素 / Use class selectors to target individual elements
- 在至少一个元素上使用 ID 选择器 / Use an ID selector on at least one element
- 在适当的地方使用标签选择器 / Use tag selectors where appropriate
- 你需要判断每个元素应该使用哪种选择器 / You will need to decide which selector type makes sense for each element

**盒模型 / Box model (LG 6.1)**
- 同时使用 margin、padding、border 和 border-box 来为至少一个组件添加样式 / Use margin, padding, border, and border-box together to style at least one component
- 你的样式应该展示你理解这些属性之间的相互作用 / Your styling should show that you understand how these properties interact with each other

---

## 提示 / Tips

- 在开始写代码之前仔细观察效果图 — 注意哪些元素具有相同的样式，哪些是独特的 / Look closely at the mockup before you start writing — notice which elements share the same style and which ones are unique
- 如果两个元素看起来一样，它们可能共享一个类 / If two elements look the same, they probably share a class
- 如果一个元素看起来是独特的，它可能有一个 ID / If one element looks unique, it probably has an ID
- 边写边在浏览器中检查你的效果 — 不需要全部完成后再预览 / Check your work in the browser as you go — you do not need to finish everything before previewing

## 评分标准 / Rubric

### LG 6.0 — CSS 选择器 / CSS Selectors

| 等级 / Level | 描述 / Description |
|---|---|
| **4 — 超越 / Exceeds** | 使用标签选择器、类选择器和至少一个 ID 选择器编写了多个规则。选择器的选择是有意且正确的 — ID 用于唯一元素，类用于共享样式的元素。学生能够通过注释或文字解释每种选择器类型的选择原因。 / Multiple rules written using tag selectors, class selectors, and at least one ID selector. Selector choices are intentional and correct. Student can explain in comments or writing why each selector type was chosen. |
| **3 — 达标 / Meets** | 正确使用了类选择器和至少一个 ID 选择器来定位和样式化单个元素。样式表中也出现了标签选择器。渲染页面与效果图非常接近。 / Class selectors and at least one ID selector are used correctly. Tag selectors also appear in the stylesheet. The rendered page closely matches the mockup. |

## 关于四级的说明 / A note on level 4

四级不仅仅意味着你的页面看起来像效果图 — 它意味着你做得更多。对于选择器，这看起来像是做出深思熟虑的选择并能够解释它们。对于盒模型，这看起来像是在多个组件上应用间距和大小，而不仅仅是任务要求的那一个。如果你的页面看起来比效果图更好，这是你在四级工作的好兆头。

A level 4 does not just mean your page looks like the mockup — it means you went further. For selectors, that looks like making deliberate choices and being able to explain them. For the box model, that looks like applying spacing and sizing across multiple components. If your page looks better than the mockup, that is a good sign you are working at level 4.
